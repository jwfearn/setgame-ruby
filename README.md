# An implementation of SET in Ruby
This code implements the [SET](http://www.setgame.com/set) card game in Ruby.  It simulates a single
game played with a shuffled deck of eighty-one cards and shows the results.

### Requirements
- Ruby

### To play:

```
$ ruby playset
```

### To test
```
$ gem install bundler
$ bundle install
$ bundle exec rspec
```
